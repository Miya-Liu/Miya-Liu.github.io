List of common commands learned from other developers: vim, bash, command line, docker.

01/06/2021 Mengya Liu Updated 1 day ago

Vim

vim filename
:qa | :w |:wq |:q! ZZ Exiting
i | A | C | u | p Editing
f[] | gg | G | L | | Navigation

Command Line

ls [option] [path] List the elements at the current folder or the give path.
tail [-F | -f | -r] [-bN | -cN | -nN] [file ...] Print last lines of each given file to a standard output.
sudo [option] "<"command">" Process commands as a root user.
wc [options] Count words by the given options.
grep [option] [pattern] [file ...] Search given files for lines that match a expression of the given pattern.

Docker

docker build | run | create | exec | start

Bash

echo Print in the command lines.
git Conditional Execution
$v Use a predefined variable whose name is 'v'.
f(){} Format of a Function
${FOO:0:3} Get the substring of variable FOO (position, length).
{5..50..5} An list of integers from 5 to 50 at the step of 5.
for ((i = 0 ; i < 100 ; i++)); do echo $i done Loop: iterate i from 0 to 99 and return i.
