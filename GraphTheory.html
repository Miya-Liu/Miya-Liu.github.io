List of Graph Problems and Algorithms.

01/06/2021 Mengya Liu Updated 1 day ago

Graph Problems and Algorithms: (related libraries: Neo4j, networkX)

Problem Computational Complexity Algorithm Optimum or Not
Breadth First Search O(n^2) O(1) stable
Depth First Search O(nlogn) O(logn) non-stable
Single Shortest Path O(n^2) Dijkstra[4], Bellman-Ford, etc stable
All Pair Shortest Path O(n^2) O(1) non-stable
Minimum Spanning Tree polynomial Borůvka[1], Prim[2], Kruskal[3] stable
Random Walk O(n^4/3) O(1) non-stable
NP-hard x non-stable
Degree Centrality O(nlogn) O(1) non-stable
Closeness Centrality O(nlogn) O(1) non-stable
Betweenness Centrality O(nlogn) O(1) non-stable
Page Rank O(nlogn) O(1) non-stable
Triangle Count O(nlogn) O(1) non-stable
Strongly Connected Components O(nlogn) O(1) non-stable
Label Propagation O(nlogn) O(1) non-stable
