Sorting Algorithms:

Note: stability refers to the performance of a sorting algorithm when it compares two elements. When there are multiple dimensions to compare the elements, such as by suit or by rank of poker cards, the correctly sorted version of an input list can have various. A stable sorting algorithm will use one dimension for its comparison, decide the equality of two elements and ignore the other dimension, at the meantime, the relative order of other dimensions will be preserved and consistent in the rest of the comparisons.
References: Sorting Algorithms Wikipedia (https://en.wikipedia.org/wiki/Sorting_algorithm#Stability).

  • Bubble Sort: switch two elements when they are in wrong order. 
    
    class Solution:
        def bubble_sort(self, nums: List[int]) -> List[int]:
            has_changed = False
            for i in range(len(nums)-1):
                has_changed = False
                for j in range(len(nums)-i-1):
                    if nums[j] > nums[j+1]:
                        tmp = nums[j]
                        nums[j] = nums[j+1]
                        nums[j+1] = tmp
                        has_changed = True
                if not has_changed:
                    break
            return nums
  • Quick Sort: divide-and-conquer, split the original list into two sub lists, namely greater one and smaller one than a pivot.
    
    class Solution:
        def quick_sort(self, nums: List[int], left, right) -> None:
            if left >= right:
                return
            i, j = left - 1, right + 1
            num = nums[(left + right) >> 1]
            while i < j:
                while True:
                    i += 1
                    if nums[i] >= num:
                        break
                while True:
                    j -= 1
                    if nums[j] <=num:
                        break
                if i < j:
                    tmp = nums[i]
                    nums[i] = nums[j]
                    nums[j] = tmp
            self.quick_sort(nums, left, j)
            self.quick_sort(nums, j+1, right)
